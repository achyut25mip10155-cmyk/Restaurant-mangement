# Restaurant-mangement
This projects shows restaurant management with dictionaries ,loop and conditional statement
this script is made to manage the customers of this restaurant .
firstly it displays menu with prizes.
accept the input and checks weather item is available or not .
then asks wheather they wnat something else.
if yes then it asks what they want .
then after all process it shows the total amount of the order.
then in last it asks that if you want to give review .
then in last they greet them.
example of menu
pizza:50
burger:50
tea:20
coffee:30
fries:30
this project is mainly made by usinf if or else statement.
Fratures :
       1,display's menu with welcoming customer
       2,item_1 if available then served else display's order unavailable
       3,another_order
            ,yes then what is order 
            ,no then order total
       4, review

Primarily this project explains the restaurent menu management with if and else statement.

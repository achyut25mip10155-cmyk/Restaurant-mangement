# Restaurant-mangement
This projects shows restaurant management with dictionaries ,loop and conditional statement
this script is made to manage the customers of this restaurant .
this python script is simple console-based restaurent ordering system that allows customers to view a menu ,order view, and get the totall amount of bill.
FEATURES:
Displays a fixed menu with item name and prices.
Takes customers input to order item from the menu.
validates if the order item is available.
Allows the customer to add additionaal order if desired.
Clculates the total price of ordered item.
Provides an option of review or end the oer with an thank you messange.

HOW TO USE:
Run the script.
The menu with prices will be displayed .
Ente the name of the item you'd like  to order.
If the item is available, it will be added to your order.
You will be asked if you want to add another item .Entre "Yes" or "No".
If yes , entre the second item to be added.
The total amount due will be displayed.
You can choose to review your order or finish.

CODE OVERVIEW:
menu:A dictionary storing items and their prices.
Users inputs to collect ordered ite.
Conditional checks to validates item availability.
Calculates and display of total order cost.
Baic flow control for adding a seecond item and order review.


POTENTIAL IMPROVEMENT:
Fix typos in menu items and input prompts.
Support multiple orders beyond two items using loops.
Make inputs case-Insensitive for better usability.
Add input validation and error handling.
Enhance user inteface for clarity.
